# Console_Snake_Cpp
A Console based snake game.It was created with an intention to learn various aspects of programming like OOPs, Functions etc .. 
## General Instruction : 
-> Use the NumPad buttions to move the snake. </br>
-> Use the following keys - </br>
<pre>
    - 8 : UP</br>
    - 4 : LEFT</br>
    - 6 : RIGHT</br>
    - 2 : DOWN</br>
</pre>
